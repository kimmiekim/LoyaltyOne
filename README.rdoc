Simple Web application that enables a user to write a post, 
make a reply to a post and 
return user's latitude, logitude and weather data based on user's current city.

Built on Rails with Angular1 and Bootstrap. 


##Instructions to open the app on your local: 

```$git clone``` 

```$cd LoyaltyOne```

```$bundle install``` 

go in to LoyaltyOne folder, and run the server 

```$rails s```



You can also find the live site on:
https://loyaltyone.herokuapp.com/
